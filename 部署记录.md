## 部署流程

### 手动部署命令

每次更新代码后，运行以下命令部署：

```bash
npm run build && rm -rf dist/.git && cd dist && git init && git add -A && git commit -m "Deploy" && git push -f https://github.com/Thomas-Liang/xuanchen_astro.git HEAD:gh-pages
```

### 注意事项

1. 访问时需要加末尾斜杠：`https://thomas-liang.github.io/xuanchen_astro/`
2. 首次访问建议使用隐私窗口避免缓存问题
3. 如遇 CSS 不加载，等待几分钟后重试
4. 从首页点击导航链接可能需要清除浏览器缓存
5. GitHub Pages 设置：Branch: gh-pages, Folder: / (root)
