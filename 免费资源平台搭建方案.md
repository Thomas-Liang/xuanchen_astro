# 免费资源分享平台搭建完整方案

## 一、需求概述

- **目标**：搭建一个免费资源分享网站，主要分享网盘等免费资源
- **规模**：大型（10个以上GitHub项目）
- **类型**：混合型（导航站 + 博客 + 后台管理）
- **技术能力**：有经验

---

## 三、技术选型

| 模块 | 推荐技术 | 理由 |
|------|----------|------|
| **静态主站** | Hugo + Vercel | 静态生成快、主题丰富、免费无限流量 |
| **带后台主站** | Linkwarden + Render | 开源、功能全、有管理后台 |
| **链接管理** | LinkStack | 自托管Linktree替代品 |
| **评论系统** | Giscus | GitHub Discussions驱动、免费 |
| **搜索** | Pagefind | 静态搜索、无需服务端 |
| **统计分析** | Umami | 可自托管、免费 |

## 六、详细部署方案

### 6.1 方案A：纯静态站点（推荐入门）

#### 目录结构

```
my-resource-platform/
├── .github/workflows/
│   └── deploy.yml
├── content/
│   ├── _index.md
│   ├── posts/                    # 博客文章
│   │   └── cloud-storage-guide/
│   │       └── index.md
│   ├── resources/               # 资源分类
│   │   ├── cloud-storage/
│   │   │   ├── _index.md
│   │   │   ├── aliyun.md
│   │   │   └── baidu.md
│   │   ├── dev-tools/
│   │   └── learning/
│   └── projects/                # GitHub项目展示
│       └── project-alpha/
├── themes/                       # 主题
├── static/                       # 静态资源
├── config.toml                   # Hugo配置
└── README.md
```

#### Hugo 配置示例 (config.toml)

```toml
baseURL = "https://your-domain.com"
title = "免费资源导航"
theme = "PaperModX"

defaultContentLanguage = "zh-cn"
languageCode = "zh-cn"

[taxonomies]
  category = "categories"
  tag = "tags"
  series = "series"

[params]
  enableSearch = true
  enableToc = true

[[menu.main]]
  name = "首页"
  url = "/"
  weight = 1

[[menu.main]]
  name = "网盘资源"
  url = "/resources/cloud-storage/"
  weight = 2

[[menu.main]]
  name = "开发工具"
  url = "/resources/dev-tools/"
  weight = 3

[[menu.main]]
  name = "项目集合"
  url = "/projects/"
  weight = 4
```

#### Vercel 部署步骤

1. 创建 GitHub 仓库
2. 登录 [Vercel](https://vercel.com)
3. Import Git Repository
4. Framework Preset 选择 Hugo
5. Deploy 完成
6. 添加自定义域名（可选）

### 6.2 方案B：带后台系统（Linkwarden）

#### 部署到 Render

##### 步骤1：Fork 仓库

```
https://github.com/linkwarden/linkwarden
```

##### 步骤2：创建 Render 账号

1. 访问 [render.com](https://render.com)
2. 用 GitHub 账号登录
3. 需要绑定信用卡（免费额度不扣费）

##### 步骤3：创建 PostgreSQL

1. Dashboard → New → PostgreSQL
2. 选择 Free 实例
3. 等待创建完成，复制 DATABASE_URL

##### 步骤4：创建 Web Service

1. New → Web Service
2. 连接到 Fork 的仓库
3. 设置环境变量：

```
DATABASE_URL=postgresql://user:pass@host:5432/db
NEXT_PUBLIC_URL=https://your-app.onrender.com
NEXTAUTH_SECRET=openssl rand -hex 32
NEXTAUTH_URL=https://your-app.onrender.com
```

4. Build Command: `npm run build`
5. Start Command: `npm run start`
6. Deploy

##### 步骤5：访问后台

- URL: `https://your-app.onrender.com`
- 首次访问自动创建管理员账号

### 6.3 子项目部署（多个GitHub项目）

#### 方案A：独立子域名（推荐）

```
your-main-site.com        # 主站
project1.your-site.com    # GitHub项目1
project2.your-site.com    # GitHub项目2
```

部署流程：
1. 每个项目独立 GitHub 仓库
2. Vercel 导入各仓库 → 自动部署
3. Cloudflare DNS 添加 CNAME
4. Vercel 设置自定义域名

#### 方案B：子目录方式

```
your-site.com/project1/
your-site.com/project2/
```

Vercel 支持配置 rewrite 规则实现。

## 七、功能实现指南

### 7.1 资源分类展示

使用 Hugo Taxonomy：

```markdown
---
title: "阿里云盘资源"
categories: ["网盘"]
tags: ["阿里云", "资源"]
---

# 阿里云盘资源汇总

## 影视资源
- [资源1](链接)
- [资源2](链接)

## 学习资料
- [资料1](链接)
```

### 7.2 链接卡片 Shortcode

创建 `layouts/shortcodes/linkcard.html`:

```html
<div class="linkcard">
  <a href="{{ .Get "url" }}" target="_blank">
    <div class="linkcard-title">{{ .Get "title" }}</div>
    <div class="linkcard-desc">{{ .Get "desc" }}</div>
  </a>
</div>

<style>
.linkcard {
  border: 1px solid #ddd;
  border-radius: 8px;
  padding: 16px;
  margin: 8px 0;
}
.linkcard-title {
  font-weight: bold;
  color: #0066cc;
}
.linkcard-desc {
  color: #666;
  font-size: 14px;
}
</style>
```

使用：
```markdown
{{< linkcard title="阿里云盘" desc="大量影视资源" url="https://..." >}}
```

### 7.3 搜索功能

安装 Pagefind：

```bash
hugo --minify
npx pagefind --site public
```

在 `layouts/_default/baseof.html` 添加：

```html
<link href="/pagefind/pagefind-ui.css" rel="stylesheet">
<script src="/pagefind/pagefind-ui.js"></script>
<div id="search"></div>
<script>
  new PagefindUI({ element: "#search" });
</script>
```

### 7.4 评论系统 (Giscus)

1. 开启 GitHub 仓库的 Discussions
2. 安装 [Giscus App](https://github.com/apps/giscus)
3. 获取配置代码
4. 添加到文章模板

## 八、域名与CDN配置

### 8.1 域名注册

| 注册商 | 特点 |
|--------|------|
| Namecheap | 首次注册便宜 |
| Cloudflare Registrar | 续费价格低 |
| 阿里云/腾讯云 | 国内备案方便 |

### 8.2 DNS 配置（Cloudflare）

1. 注册 Cloudflare 账号
2. 添加域名
3. 修改域名服务器
4. 添加 DNS 记录：

```
类型    名称              值
A       @                 76.76.21.21 (Vercel)
CNAME   www               cname.vercel-dns.com
CNAME   project1          cname.vercel-dns.com
```

### 8.3 免费SSL

Cloudflare 自动提供 Universal SSL
