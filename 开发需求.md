## 技术选型

| 模块 | 技术 | 理由 |
|------|------|------|
| 静态站点框架 | Astro | 支持多框架组件、Islands架构、零JS默认 |
| 部署平台 | GitHub Pages + Actions | 免费、自动部署 |
| 评论系统 | Giscus | GitHub Discussions驱动 |
| 数据库 | SQLite + Drizzle ORM | 本地轻量，无需外部服务 |
| 后台框架 | Astro SSR | 前后端一体化 |
